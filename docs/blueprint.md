## Core Features:

- Dashboard Overview: Display key financial metrics such as total net worth, cash flow, assets, and debts.
- Transaction Tracking: Allow users to view and categorize their transactions.
- Investment Tracking: Enable users to track their investments and portfolio performance.
- Debt Management: Provide tools for managing and tracking debts, including due dates and amounts.
- Financial Reports: Generate reports and visualizations of financial data.
- Asset Allocation Visualization: Display asset allocation using a pie chart.
- Exchange rate analysis: Get recommendations about financial news to read about exchange rate opportunities using the AI to analyze market data and financial news.

## Style Guidelines:

- Primary color: Dark charcoal (#121212) for a modern, sophisticated feel, reminiscent of dark mode interfaces.
- Background color: Almost-black (#0A0A0A), only slightly lighter than the primary, to maintain a stark dark mode style.
- Accent color: Bright green (#00FF44) for highlighting key metrics and interactive elements, ensuring contrast and visibility.
- Font: 'Inter', a sans-serif font, for both headlines and body text. 'Inter' offers a modern, machined, neutral look suitable for a financial dashboard.
- Use simple, outlined icons in green (#00FF44) to represent different categories and actions.
- Design a clean, card-based layout with clear separation between sections.
- Incorporate subtle transitions and animations to provide feedback on user interactions.
